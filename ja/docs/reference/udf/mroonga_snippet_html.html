5.7. mroonga_snippet_html()¶

バージョン 5.09 で追加.

5.7.1. 概要¶

mroonga_snippet_html() UDF provides functionality to get highlighted keyword in context.

注釈

この機能は実験的です。引数や引数の値が予告なく変更される可能性があります。
