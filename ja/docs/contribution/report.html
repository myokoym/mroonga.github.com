7.1. 見つけたバグを報告するには¶

バグを報告する方法は2つあります。:

  • Issue trackerにバグを報告する
  • メーリングリストへバグを報告する

どちらのやりかたを使ってもらっても構いません。

7.1.1. Issue trackerにバグを報告する¶

Mroongaプロジェクトでは、GitHub issue tracker を使っています。

バグを報告するのは英語でも日本語でもどちらでも良いです。

7.1.2. メーリングリストへバグを報告する¶

Mroongaプロジェクトでは、Mroongaについて話す コミュニティ があります。バグの内容を記載したEメールを送ってください。
